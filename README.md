# 🚀 Projeto de Automação Cypress

Projeto de automação de testes E2E utilizando Cypress, simulando cenários de cadastro de candidatos em formulário web.

---

## 📋 Sobre o projeto

Este projeto foi desenvolvido com foco em aprendizado e prática de automação de testes utilizando Cypress.

Os testes validam:
- preenchimento de formulário
- validações de campos
- mensagens de erro
- fluxo de cadastro
- validação de email
- proteção contra dados inválidos

---

## 🛠 Tecnologias utilizadas

- Cypress
- JavaScript
- Node.js
- GitHub Actions
- Git
- CI/CD

---

## ✅ Cenários automatizados

- Cadastro com sucesso
- Campos obrigatórios
- Validação de nome
- Validação de sobrenome
- Validação de email inválido
- Validação de senioridade
- Proteção contra scripts maliciosos
- Clique duplo no botão

---

## ▶️ Como executar o projeto

### Instalar dependências

```bash
npm install
```

### Abrir Cypress

```bash
npx cypress open
```

### Executar em modo headless

```bash
npx cypress run
```

## 🔄 Integração Contínua

Projeto configurado com GitHub Actions para execução automática dos testes a cada push realizado no repositório.
